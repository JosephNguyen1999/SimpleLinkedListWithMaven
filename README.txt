This README txt file is for the project4 directory.

To compile this project:

You must first be in the project4 directory and then type:

    $ mvn compile

The resulting files will be in the target subdirectories:

   ./target/classes         -- your source code class files
   ./target/test-classes    -- your test classes class files

To test the project with JUnit:

   $ mvn test

All Test classes will be executed in JUnit and the report will be listed.

To generate the JavaDoc documentation:

   $mvn site

The resulting JavaDoc files will be in:

    ./target/site
